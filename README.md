# LatLens 🌍

LatLens is a Chrome extension for converting geospatial coordinates between Decimal Degrees (DD) and Degrees-Minutes-Seconds (DMS), with map preview.

## Features
- DD ↔ DMS conversion (mathematical)
- Map preview 
- Copy results
- Persistent state (Chrome Storage)
- Privacy-safe (no page access)

## Use Cases
- Convert coordinates for mapping or GIS tasks  
- Validate and format location data  
- Quickly visualize coordinates on a map  

## Target Users
- Developers working with location data  
- GIS / mapping enthusiasts  
- Students learning geospatial concepts  

## Tech Stack
- React (Vite)
- Tailwind CSS
- Chrome Extension (Manifest V3)
